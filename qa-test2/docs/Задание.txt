Описание:
Реализована система предоставления данных по пациентам за различные даты и из разных источников.
При нажатии на «Построить отчет» выводится таблица с данными в соответствии с выбранными критериями.
В системе предусмотренно оповещение поьзователя при некорретном выборе комбинации полей даты и источника.
Выгрузка данных таблицы в csv-файл реализована с помощью готовой и протестированной библиотеки, результат выгрузки по-умолчанию считается корректным.

В построенной таблице данные заполняются в соответствии с условиями:
•	В столбце «Данные1» значения не могут превышать 80.
•	В столбце «Данные2» значения не могут превышать 100.
•	В столбце «Данные3» значения должны быть не меньше 5.
•	В столбце «Итог» отображается сумма значений по каждому столбцу.
•	Для каждого пациента значения анализов не могут совпадать за разные даты и из разных источников.
•	Таблица не может cодержать пустых полей.
•	Количество пациентов в таблице не может превышать 30.

Необходимо:
1.	Протестировать систему и составить баг-репорты на найденные дефекты.
2.	Составить тест-кейсы для проверки функционала системы.
